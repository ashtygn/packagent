# demo/ — the open pkgtk demo (PHOTON-X1)

Everything here is open-source-safe: the package is **PHOTON-X1**, a fully synthetic
2025-ball (45×45, 0.8 mm pitch) design generated by `pkgtk.synth` with a fixed seed —
byte-identical on every regeneration. The physics receipt is a plane-pair structure
built 100% from scratch in pyedb and solved headless by Ansys SIwave; the committed
Touchstone files are that solver's real output. No customer or vendor design data
appears anywhere in this tree.

## Contents

| Dir | What | Run |
|---|---|---|
| `pkg/` | PHOTON-X1 generator + seeded-defect / ECO-revB / spreadsheet emitters | `python pkg/gen_photonx1.py` → graph + runconfig JSON; then `make_defect.py`, `make_revb.py`, `make_pinmap_xlsx.py` |
| `geom/` | clean + dirty synthetic substrate GDS generator (4 planted defects) | `python geom/gen_substrates.py`, then `pkgtk check` per docs/DEMO_SCRIPT.md |
| `phys/` | the SIwave receipt: EDB build (`t1_create.py`), ports+SYZ (`t5_syz.py`), 12 GHz extended solve (`t6_extend_12g.py`), and the cavity-vs-solver comparison (`cavity_vs_solver.py`) + **real solver Touchstones** (`plane_pair_touchstone.s2p`, `ext12g/plane_pair_12g_touchstone.s2p`) | `python phys/cavity_vs_solver.py phys/ext12g/plane_pair_12g_touchstone.s2p` — needs only numpy + pkgtk (no license); the t-scripts need pyedb + an Ansys install |
| `graphs/` | matplotlib figures generated from the real data (cavity-vs-SIwave overlay with peak deltas, the 2025-ball map, pkgtk's own PDN plot) | `python graphs/make_graphs.py` (regenerates the PNGs) |
| `dashboard/` | self-contained animated presenter dashboard (`index.html`, 70 KB, zero network/CDN) — 8 keyboard-driven panels rendering the embedded real data | open `index.html` in a browser, F11; ←/→ advance, 0–7 jump, `?still` disables animations |

Talk track and choreography: `../docs/DEMO_SCRIPT.md` + `../docs/PRESENTATION.md`.
Agent operating manual (the whole pipeline is headless-driveable): `../AGENTS.md`.

## Headline results (all reproduced live, wall times measured)

- 2025-ball package generated in ~20 ms; clean verify = **0 violations** (the
  false-positive gate); seeded defect caught exactly (`bijection.multi_assignment @ A11`).
- Seeded-defect benchmark: **20/20 caught, 0 false positives** (`python -m benchmarks.run`).
- Spreadsheet cross-check: clean = 0 findings; 3 seeded drift errors caught by cell.
- ECO diff: exactly `{removed: A11_N, pair_broken: A11_P}`.
- Geometry: clean GDS = 0; dirty GDS = **4/4 planted defects** with exact µm and
  clickable KLayout markers; coverage table shows unimplemented checks loudly.
- Physics: analytic cavity oracle vs SIwave on the same structure — resonance peaks
  within **3.68% / 1.00%**; low-frequency agreement 95.6% after the documented
  fringing correction; the solver's own Touchstone **passes pkgtk's physics gate**
  (passivity good, reciprocity exact).

## Deliberately NOT committed

- Cadence sample-design derivatives (`.sip`/`.mcm` marker databases, APD journals) —
  Cadence install content is not redistributable; the demo *shows* them locally only.
- Solver project binaries (`.aedb`, `.siw`, results dirs) — regenerable via the
  t-scripts; only the Touchstone outputs (plain text, our own synthetic structure)
  are committed as the receipt.
- Anything from a real/customer design. See `AGENTS.md` non-negotiables.
